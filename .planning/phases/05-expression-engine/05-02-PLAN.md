Add to `packages/core/expression.ts`:

**ExpressionContext type:**
```typescript
export type ExpressionContext = Record<string, unknown>;
```

**extractExpression(template: string): string:**
- Trim the input
- Verify it starts with `{{` and ends with `}}`
- Return the inner content trimmed
- Throw ExpressionError if delimiters missing

**evaluateNode(node: ASTNode, context: ExpressionContext): unknown:**
Tree-walk evaluator with switch on `node.type`:

- **Literal**: return `node.value`
- **Identifier**: return `context[node.name]`
- **MemberExpression**: evaluate object, then access property. If object is null/undefined, return undefined (safe navigation -- do NOT throw). For `.length` on arrays, return `.length`.
- **BinaryExpression**: evaluate left and right, then dispatch on operator:
  - `==`: loose equality (`==`)
  - `!=`: loose inequality (`!=`)
  - `<`, `>`, `<=`, `>=`: numeric/string comparison
  - `&&`: SHORT-CIRCUIT -- evaluate left first, if falsy return left, else evaluate right
  - `||`: SHORT-CIRCUIT -- evaluate left first, if truthy return left, else evaluate right
- **UnaryExpression**: `!` -> negate truthiness of operand
- **ConditionalExpression**: evaluate test, if truthy evaluate consequent, else evaluate alternate
- **CallExpression**: evaluate callee, then dispatch on `node.property`:
  - Verify target is an array, throw ExpressionError if not
  - `"contains"`: map to `Array.prototype.includes(evaluatedArgs[0])`
  - `"every"`: if no args, check all elements truthy (`target.every(Boolean)`). If one string arg, check named property on each element (`target.every(item => (item as any)[propName])`)
  - `"some"`: same pattern as every but with `.some()`
  - Any other method: throw ExpressionError("Unknown method: .{method}()")

**evaluate(template: string, context: ExpressionContext): unknown:**
```typescript
export function evaluate(template: string, context: ExpressionContext): unknown {
  const expr = extractExpression(template);
  const tokens = tokenize(expr);
  const ast = parse(tokens);
  return evaluateNode(ast, context);
}
```
