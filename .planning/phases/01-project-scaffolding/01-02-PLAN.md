  <name>Task 1: Create GitHub Actions CI workflow</name>
  <files>
    .github/workflows/ci.yml
  </files>
  <action>
    Create `.github/workflows/ci.yml` with the following configuration:

    ```yaml
    name: CI

    on:
      pull_request:
        branches: [main, develop]

    jobs:
      check:
        runs-on: ubuntu-latest
        steps:
          - uses: actions/checkout@v4
          - uses: actions/setup-node@v4
            with:
              node-version: 20
              cache: npm
          - run: npm ci
          - run: npm run lint
          - run: npm run typecheck
          - run: npm test
    ```

    Key details:
    - Triggers on pull_request to both main AND develop branches
    - Uses Node 20 (LTS, above the Node 18+ minimum target)
    - Caches npm dependencies for faster CI runs
    - Runs npm ci (clean install) not npm install
    - Runs lint, typecheck, and test in that order (fail fast on cheapest checks)
    - Single job is sufficient for this project size
  </action>
